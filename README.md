# PACRO - Parking Application Croatia 📱</br></br>
Mobile app for parkings made with ```ionic, vue, firebase and electron.```</br></br>
 Android📱 - run the Android folder in android studio with you mobile phone</br></br>
 Desktop🖥️ - run the adminpaneldesktop.exe in ..\adminpaneldesktop\dist_electron</br></br>
 For web preview of the app run ionic serve and for the admin panel open the adminpaneldesktop folder and run npm run electron:serve.
